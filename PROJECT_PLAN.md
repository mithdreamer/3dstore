# Project Plan

## Tamamlanan ilk surum

- Statik ana sayfa ve Netlify uyumlu kok yonlendirme
- JSON tabanli 8 urun ve 5 kategori
- Kategori filtreli, aramali ve siralamali urun katalogu
- Dinamik urun detay sayfasi
- localStorage tabanli sepet
- Demo checkout ve yerel siparis kaydi
- Hakkimizda ve iletisim sayfalari
- Ana sayfa icin bitmap hero gorseli

## Teknik tercih

Backend olmadan, sadece HTML/CSS/JavaScript ile baslandi. Bu sayede GitHub'a push edildikten sonra Netlify uzerinde build komutu olmadan yayinlanabilir.
